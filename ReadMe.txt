-- One Man Gang --

The game is played using an Xbox 360 Controller

Software Requirements:
Built using SFML 2.4.2
with an Environment path for SFML_SDK

Controls:
Move - Left Analog Stick
Jump - A Button
Shoot -Right Trigger

Instructions:
Collect pickups to upgrade your guns for a limited
time before getting put back to your Pistol

Kill enemies and see how many waves you can last for

-- Enjoy the Game --
